Laboratorio de Datos

Repositorio de datasets y API construido a partir de datos facilitados por entidades de gobierno al Laboratorio para la Ciudad, estructurados por hackers y plataformas cívicas.

Aquí encontrarás recursos digitales para desarrollar proyectos, así como tutoriales que te ayudarán a entender y usar los datasets.

Para más información acerca de los formatos de participación y entregables, revisa la sección de Retos en hack.labcd.mx.
