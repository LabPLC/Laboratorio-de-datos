Acerca de Laboratorio de Datos

El Laboratorio de Datos es un repositorio de datasets y API construido a partir de datos facilitados por entidades de gobierno al Laboratorio para la Ciudad, estructurados por hackers y plataformas cívicas.

El Laboratorio de Datos es el repositorio base de HackCDMX, festival de Datos de la Ciudad de México y CódigoCDMX, ambos proyectos de innovación cívica del Laboratorio para la Ciudad.

El Laboratorio para la Ciudad es el área experimental de innovación cívica y creatividad urbana del Gobierno de la Ciudad de México.
